# Show

Returns the details of specified model.

## Usage

```bash copy
curl -X GET 'http://127.0.0.1:5000/show?model=mlx-community/mistral-7B-v0.1'
```

## Example

**Python**

```python copy
from mlx_server import MLXServer

server = MLXServer(model="mlx-community/mistral-7B-v0.1")

```

**Curl**

```bash copy
curl -X GET 'http://127.0.0.1:5000/show?model=mlx-community/mistral-7B-v0.1'
```

## Parameters

**model** - The model you want to show details about.
