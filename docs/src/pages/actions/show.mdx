# Show

Returns the details of specified model.

## Usage

```python copy
server.show(model="mlx-community/mistral-7B-v0.1")
```

## Example

```python copy
from mlx_server import MLXServer

server = MLXServer(model="mlx-community/mistral-7B-v0.1")

server.show("mlx-community/mistral-7B-v0.1")
```
