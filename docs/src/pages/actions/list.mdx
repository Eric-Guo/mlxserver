# List

A list of all the locally downloaded models.

## Usage

```bash copy
curl -X GET 'http://127.0.0.1:5000/list'
```

## Example

**Python**

```python copy
from mlxserver import MLXServer

server = MLXServer(model="mlx-community/Nous-Hermes-2-Mistral-7B-DPO-4bit-MLX")
```

**curl**

```bash copy
curl -X GET 'http://127.0.0.1:5000/list'
```
